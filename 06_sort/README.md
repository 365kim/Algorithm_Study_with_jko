## 6. Sort
- n개의 데이터가 주어졌을 때, 사용자가 정한 기준에 맞게 정렬하여 출력하는 알고리즘
- 효율적인 정렬은 다른 알고리즘을 최적화하는데 중요한 역할을 함
- 정렬의 종류
    - __단순 정렬__ : 삽입정렬, 선택정렬
    - __효율적인 정렬__ : 힙 정렬, 퀵 정렬, 합병 정렬, 셸 소트
    - __거품정렬 및 변종__ : 거품 정렬, 빗질 정렬
    - __분산 정렬__ : 계수 정렬, 버킷 정렬, 기수 정렬 등
