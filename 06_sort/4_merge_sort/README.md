## Merge Sort
- __합병정렬 알고리즘__
1. 리스트를 균등하게(1/2) 둘로 쪼갬
2. 분할된 두 리스트의 정렬 결과를 임시리스트에 저장
3. 임시리스트에 저장된 정렬내용을 원본 리스트에 복사
4. 크기가 1인 리스트부터 정렬된 리스트를 재귀적으로 합병하면서 전체정렬 완성
    - 퀵소트와 마찬가지로 분할정복(Divide & Conquer) 방식
