## Bubble Sort
- 가장 간단하지만 가장 비효율적인 정렬 알고리즘
- 첫번째 항목부터 마지막항목까지 서로 인접한 두 원소를 검사하고 필요시 swap
- 작은 원소가 맨 끝으로 이동해서 정렬되는 모습이 거품이 떠오르는 것 같다해서 버블정렬이라고 부름
- 다른 정렬과 달리 이미 정렬되어있는지 여부를 알 수 있음
