### Definition
- 데이터를 효율적으로 이용할 수 있도록 컴퓨터에 저장하는 방법
- 데이터의 특성에 따라 분류하여 구성하고 저장 및 처리하는 모든 작업을 의미
- 데이터 처리의 성능과 효율에 직접적으로 영향을 끼치기 때문에 실제 데이터들의 관계를 잘 반영해야 함
- 이렇게 자료구조로 표현된 데이터들을 이용하여 데이터들을 처리하는 절차들이 `알고리즘`
