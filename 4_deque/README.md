## 4. Deque
- 양쪽 끝 데이터 삽입/삭제가 모두 가능한 자료구조
- deque 는 'double-ended queue'를 의미함
- 큐, 스택의 기능을 합친 형태로 자유도가 높음
- `front` : 덱의 맨 앞, `back` : 덱의 맨 뒤
- `push_front`, `push_back` : 데이터 삽입연산
- `pop_front`, `pop_back` : 데이터 삭제연산
- `begin`, `end` : peek연산
