## 5. Tree
- 나무를 뒤집어놓은 모양의 노드들로 이루어진 비선형적 자료구조로, 데이터가 계층적으로 이루어져 있음
- `root` : 최상단 노드
- `leaf` : 최하단 노드
- `child` : 링크로 연결된 하위 노드
- `parent` : 링크로 연결된 상위 노드
- `subtree` : child를 root로 하는 트리
- `degree` : child 노드의 수
- `depth` : root 기준, `height` : leaf 기준
- `binary tree` : 각각의 노드가 최대 2개의 child 노드를 갖는 트리
